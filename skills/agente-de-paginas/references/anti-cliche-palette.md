# Anti-clichê — referência rápida

A regra de ouro: se você chegou em um destes três por padrão (não porque o cliente pediu), revise. São legítimos *se o brief pedir exatamente isso*.

## Clichê 1 — Creme + serifada + terracota

**Look**: fundo creme (~#F4F1EA), serifada de alto contraste (Playfair, Fraunces, EB Garamond), acento terracota (~#C66B3D). Muito "revista lifestyle". Funciona pra um nicho específico (moda autoral, gastronomia artesanal, editorial de hotelaria boutique). Não funciona como default para clínica médica, SaaS B2B, e-commerce de suplementos, fintech etc.

**Como substituir** quando o cliente não é lifestyle/editorial:
- Fundo em azul-noite profundo + tipografia geométrica + acento em âmbar (tech/fintech sério).
- Off-white quente (#FAF7F2) + grotesca industrial + acento em oliva escuro (consultoria, arquitetura).
- Cinza-grafite + grotesca mono como display + acento em rosa-choque (produto criativo, plataforma para designers).

## Clichê 2 — Preto + neon único

**Look**: fundo ~#0A0A0A, texto branco, um único acento neon (verde-ácido ou vermelhão). Funciona pra produto tech dev-friendly, ferramenta para devs, cripto, gaming. Não funciona como default para coach, e-commerce de moda feminina, restaurante, advocacia.

**Como substituir** quando o cliente não é tech/dev/cripto:
- Bege profundo (#1A1612) + tipografia mista (serifada display + sans-ui) + acento em dourado fosco (premium, joalheria, hotelaria de luxo).
- Marrom-argila (#2B1E18) + grotesca neutra + acento em verde-musgo (cafeteria especialidade, destilados, mobiliário).
- Azul-petróleo escuro (#0D2230) + tipografia humanista + acento em coral suave (wellness, ioga, saúde integrativa).

## Clichê 3 — Jornal com hairlines, zero border-radius, colunas densas

**Look**: inspiração New York Times / Bloomberg, tipografia serifada de notícia, divisores finíssimos de 1px, blocos justificados, grid rígido de 12 colunas, zero arredondamento. Funciona pra jornal digital, newsletter financeira, consultoria estratégica, fundo de hedge fund. Não funciona como default para D2C, infoproduto, marca de beleza, comunidade.

**Como substituir** quando o cliente não é editorial/financeiro/jornal:
- Cards com border-radius generoso (12–20px) + sombra suave + tipografia humanista (infoproduto, comunidade, D2C).
- Layout assimétrico com colunas desiguais + tipografia mista (display bold + body leve) para marca criativa.
- Estrutura "tabs" ou "stages" com cor de seção mudando a cada bloco (SaaS, plataforma de educação).

## Pergunta de calibragem

Antes de aprovar a paleta, responda: "se eu trocasse o nome do cliente por outro completamente diferente do mesmo porte, essa página ainda faria sentido?" Se sim, a paleta ainda é genérica — volte e ajuste.
